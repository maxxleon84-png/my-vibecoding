| Роль | Переменная CSS | HEX | Пример использования |
|------|---------------|-----|----------------------|

| Текст синеватый | `--color-text-muted` | `#6B9ECC` | Второстепенный текст, подписи |
| Glow-синий | `--color-glow` | `#0055CC` | Box-shadow, text-shadow эффекты |

## Эффекты и детали

### Glow (свечение)
```css
/* Синее свечение на кнопках/элементах */
box-shadow: 0 0 20px rgba(0, 119, 255, 0.4);

/* Циановое свечение (акцент) */
box-shadow: 0 0 15px rgba(0, 212, 255, 0.5);

/* Text glow для заголовков */
text-shadow: 0 0 20px rgba(0, 212, 255, 0.6);
```

### Градиенты
```css
/* Hero-градиент (фон) */
background: linear-gradient(135deg, #070D1F 0%, #0D1B3E 50%, #070D1F 100%);

/* Кнопка primary */
background: linear-gradient(135deg, #0077FF, #00D4FF);

/* Overlay для секций */
background: linear-gradient(180deg, transparent, rgba(7, 13, 31, 0.9));
```

### Анимации
- Пульсирующий glow на кнопках (keyframe `pulse`)
- Подчёркивание с циановым цветом на hover ссылок
- Плавное появление карточек (fade-in + slide-up)

---

## Правила применения

1. **Никогда** не используй белый фон — только тёмные оттенки из палитры
2. Текст на тёмном фоне — всегда белый или `--color-text-muted`
3. CTA-кнопки — всегда с gradient primary + glow эффект
4. Иконки и акценты — `--color-accent` (#00D4FF)
5. Все три продукта (визитка, лендинг, квиз) используют одну эту палитру
