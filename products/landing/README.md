# Продукт: Лендинг

**Статус:** 📋 В плане
**Папка:** `products/landing/`

---

## Цель
Конвертировать посетителей в заявки через Telegram.

## Секции (по порядку)

1. **Hero** — фото, имя, главный оффер, кнопка «Написать в TG»
2. **Услуги** — карточки: визитка, лендинг, квиз (цена, срок)
3. **Как работаю** — 4 этапа (бриф → разработка → правки → сдача)
4. **Портфолио** — галерея скриншотов/ссылок на работы
5. **Отзывы** — карточки с отзывами клиентов
6. **Квиз** — встроенная секция «Подберём продукт для тебя»
7. **Контакт** — ссылки на Telegram, email, соцсети

## Файлы (создать в Этапе 3)

```
products/landing/
├── index.html
├── styles.css
├── script.js
└── README.md  ← ты здесь
```

## Технические требования

- Подключить `../../shared/css/variables.css`
- Плавная прокрутка между секциями
- Квиз встроен как `<section id="quiz">`
- Mobile-first, breakpoints: 768px, 1200px
- Все CTA кнопки → Telegram

## Важно
Квиз из `products/quiz/` переиспользуется здесь как встроенная секция.
