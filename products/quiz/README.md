# Продукт: Квиз (лидогенерация)

**Статус:** 📋 В плане
**Папка:** `products/quiz/`

---

## Цель
Квалифицировать посетителя → дать рекомендацию → получить заявку в Telegram.

## Логика

```
Вопрос 1: Что нужно?
  → Визитка / Лендинг / Квиз / Не знаю

Вопрос 2: Есть материалы?
  → Да / Частично / Нет

Вопрос 3: Сроки?
  → Срочно / Не срочно

РЕЗУЛЬТАТ:
  → Рекомендация продукта
  → Кнопка «Написать в Telegram»
```

## UX-требования

- Один вопрос на экране (шаг за шагом)
- Прогресс-бар (шаг 1/3, 2/3, 3/3)
- Анимация перехода между вопросами
- На результате: иконка + название продукта + описание + CTA кнопка

## Файлы (создать в Этапе 4)

```
products/quiz/
├── index.html     (standalone версия)
├── styles.css
├── script.js      (логика квиза)
└── README.md      ← ты здесь
```

## Встраивание в лендинг
Код квиза переиспользуется в `products/landing/index.html` как секция `<section id="quiz">`.
Логика из `script.js` подключается в `products/landing/script.js`.

## Технические требования

- Подключить `../../shared/css/variables.css`
- Чистый JS без библиотек
- Работает как standalone (`products/quiz/index.html`) и как встроенный модуль
- Все результаты ведут в Telegram https://t.me/maxxleon927
